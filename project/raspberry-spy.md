A Raspberry Pi with a camera module and PIR motion sensor. When the PIR triggers, it captures a still and emails it out. Originally built to keep an eye on an empty apartment; still a useful reference for anyone wiring sensors to a Pi.

Source on GitHub: [cthare/Raspberry-Spy](https://github.com/cthare/Raspberry-Spy).
